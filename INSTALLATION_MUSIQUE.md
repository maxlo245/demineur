# ?? Installation des Musiques - Kevin MacLeod

## Guide Rapide d'Installation

### Étape 1 : Créer le dossier
1. Créez un dossier `sounds` dans le répertoire du jeu (même niveau que l'exe)
2. Structure finale :
   ```
   demineur/
   ??? demineur.exe
   ??? sounds/
       ??? menu.mp3
       ??? game.mp3
       ??? win.mp3
       ??? (optionnel) fichiers .wav pour les SFX
   ```

## ?? Test

1. Lancez le jeu
2. Appuyez sur **M** pour activer/désactiver l'audio
3. Si la musique ne joue pas :
   - Vérifiez que les fichiers sont dans `sounds/`
   - Vérifiez les noms de fichiers (sensible à la casse)
   - Essayez de convertir en .wav si vous avez des MP3

## ?? Dépannage

**La musique ne joue pas :**
- ? Vérifiez que le dossier `sounds/` existe
- ? Vérifiez les noms de fichiers (pas de majuscules)
- ? Essayez format .wav au lieu de .mp3
- ? Vérifiez que l'audio n'est pas désactivé (touche M)

**La musique coupe ou saute :**
- Réduisez la qualité/taille du fichier
- Utilisez un format .wav plus stable
- Vérifiez que votre PC n'est pas surchargé

**Pas de son système :**
- Vérifiez le volume Windows
- Vérifiez que le périphérique audio est activé
