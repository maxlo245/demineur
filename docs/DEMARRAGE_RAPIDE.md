# ?? DÉMARRAGE RAPIDE - Démineur HD

## 3 Étapes Simples pour Jouer

### Étape 1 : Lancer le Jeu (Sans Musique) ?
```bash
1. Double-cliquez sur demineur.exe
2. Jouez ! (Sons système Windows)
```

### Étape 2 : Ajouter les Musiques (Optionnel) ??
```bash
1. Double-cliquez sur telecharger_musiques.bat
2. Suivez les instructions à l'écran
3. Téléchargez les 3 musiques
4. Renommez-les et placez-les dans sounds/
```

### Étape 3 : Vérifier l'Installation ??
```bash
1. Double-cliquez sur verifier_audio.bat
2. Vérifiez que tout est OK
3. Relancez le jeu !
```

---

## ?? Contrôles du Jeu

### Souris
- **Clic gauche** ? Révéler une case
- **Clic droit** ? Poser/enlever un drapeau

### Clavier
- **H** ? Utiliser une aide (3 max)
- **P** ? Pause
- **R** ? Recommencer
- **M** ? Activer/désactiver l'audio
- **ESC** ? Retour au menu

## ?? Conseils pour Débutants

### 1?? Commencez par les coins
Les coins sont généralement plus sûrs pour le premier clic.

### 2?? Utilisez les chiffres
- **1** = 1 mine adjacente
- **2** = 2 mines adjacentes
- etc.

### 3?? Drapeaux = Votre ami
Marquez les mines avec des drapeaux (clic droit) pour ne pas cliquer dessus par erreur.

### 4?? Utilisez les hints
Appuyez sur **H** pour révéler une case sûre quand vous êtes bloqué (3 hints max).

### 5?? Visez les combos
Révélez plusieurs cases d'un coup pour obtenir un multiplicateur de score !

---

## ?? Système de Score

Votre score est calculé selon :
- ? **Cases révélées** : 10 points par case
- ? **Vitesse** : Bonus si vous êtes rapide
- ? **Combo** : Multiplicateur jusqu'à x5
- ? **Hints non utilisés** : 50 points par hint restant

**Objectif : Maximiser votre score en étant rapide et précis !**

## ?? FAQ Audio

### La musique ne joue pas ?
1. Vérifiez que les fichiers sont dans `sounds/`
2. Vérifiez les noms : `menu.mp3`, `game.mp3`, `win.mp3`
3. Appuyez sur **M** dans le jeu
4. Lancez `verifier_audio.bat`

### Je n'ai que des .wav et pas de .mp3 ?
? Pas de problème ! Le jeu accepte les deux formats.

### Puis-je utiliser mes propres musiques ?
? Oui ! Renommez-les selon les noms attendus et placez-les dans `sounds/`.

## ?? Problèmes Courants

### Le jeu ne se lance pas
- ? Vérifiez que vous avez **Windows 7 ou supérieur**
- ? Installez les **Visual C++ Redistributables**
- ? Exécutez en tant qu'administrateur

### Performance lente
- ? Fermez les applications gourmandes
- ? Le jeu cible 60 FPS en 1920×1080
- ? Vérifiez les pilotes graphiques

### Pas de son du tout
- ? Vérifiez le volume Windows
- ? Vérifiez que le périphérique audio est activé
- ? Appuyez sur **M** dans le jeu
